# dot-vimrc

## Installing

1. clone this repo
1. create symlink for folder `ln -s ~/dev/dot-vimrc/ ~/.vim`
1. create symlink for vimrc `ln -s ~/dev/dot-vimrc/.vimrc ~/.vimrc`
1. load vim and run `PlugInstall`
